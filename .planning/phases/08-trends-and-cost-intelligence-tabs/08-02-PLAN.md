**3. Add cost calculation helper functions (before renderCost):**

`calcAvgUnitPrice(groupId)` — compute average unit price for an item group:
- Find all MOCK_PURCHASE_ITEMS with matching groupId
- For each, find all MOCK_PURCHASES with that purchaseItemId
- Compute weighted average price: sum(price * qty) / sum(qty) across all purchases
- Return the average (or 0 if no purchases)

`calcMenuItemCost(menuItem)` — compute estimated cost for a menu item (per D-10):
- For each groupId in menuItem.ingredientGroupIds, get calcAvgUnitPrice(groupId)
- Each ingredient contributes an equal proportion: 1 / ingredientGroupIds.length
- Estimated cost = sum of (avgUnitPrice * proportion) for each ingredient
- Return {totalCost, ingredients: [{name, groupId, proportion, avgUnitPrice, lineCost}]}

`calcIngredientUsage(groupId)` — compute which menu items use this ingredient (per D-09):
- Find all MOCK_MENU_ITEMS where ingredientGroupIds includes groupId
- For each menu item, compute:
  - costContribution: calcAvgUnitPrice(groupId) * (1/menuItem.ingredientGroupIds.length)
  - totalSalesQty from MOCK_SALES
  - revenue: totalSalesQty * menuItem.price
  - returnOnPurchase: revenue / costContribution (as a ratio)
- Return array of {menuItemName, costContribution, revenue, returnOnPurchase, pct} where pct is this ingredient's share of total cost contributions across all menu items that use it
