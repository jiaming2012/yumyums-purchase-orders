    txt:s.getPropertyValue('--txt').trim(),

`spendByTag(chips)` — aggregate total spending per tag, filtered by active chips:
- Build purchaseItemId -> groupId lookup from MOCK_PURCHASE_ITEMS
- Build groupId -> tagIds lookup from MOCK_ITEM_GROUPS
- Iterate MOCK_PURCHASES, find the tag for each purchase via the chain: purchase.purchaseItemId -> groupId -> tagIds[0]
- Sum price*qty per tag
- If chips Set is non-empty, filter to only those tag IDs
- Return {labels:[], data:[], colors:[]} using TAG_EMOJI + tag.name for labels and palette colors

`spendByMonth(chips)` — aggregate total spending per month key (YYYY-MM format per Pitfall 8):
- Use parseLocalDate() on each MOCK_PURCHASE_EVENT date (never raw new Date per Pitfall 4)
- Build purchaseItemId -> groupId -> tagIds chain for filtering
- For each purchase, get its event date, compute YYYY-MM key
- If chips Set is non-empty, only include purchases whose item group has a matching tagId
- Sum price*qty per month key
- Sort keys alphabetically (correct chronological order for YYYY-MM)
- Format labels as "Jan 2026" style via Intl.DateTimeFormat
- Return {labels:[], data:[]}
