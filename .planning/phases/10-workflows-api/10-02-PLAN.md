Create `backend/internal/workflow/repository.go` in package `workflow`. Follow the query pattern from `me/handler.go` (pool.Query, row scanning, pgxpool.Pool as parameter). All functions take `ctx context.Context` and `pool *pgxpool.Pool` as first two args.

**Template CRUD:**

1. `func insertTemplate(ctx context.Context, pool *pgxpool.Pool, input TemplateInput, createdBy string) (string, error)` — BEGIN transaction. INSERT into checklist_templates RETURNING id. For each section: INSERT into checklist_sections RETURNING id. For each field: INSERT into checklist_fields. For sub-steps (FieldInput.SubSteps): insert parent field first, get its ID, then insert sub-step fields with parent_field_id. Insert schedules and assignments. COMMIT. Return template ID.

2. `func replaceTemplate(ctx context.Context, pool *pgxpool.Pool, templateID string, input TemplateInput) error` — BEGIN transaction. DELETE FROM checklist_sections WHERE template_id = $1 (cascades to fields). DELETE FROM checklist_schedules WHERE template_id = $1. DELETE FROM template_assignments WHERE template_id = $1. Re-insert all sections, fields, schedules, assignments (same as insertTemplate inner loop). UPDATE checklist_templates SET name=$1, requires_approval=$2, updated_at=now(). COMMIT. Per D-09 full replace.

3. `func archiveTemplate(ctx context.Context, pool *pgxpool.Pool, templateID string) error` — UPDATE checklist_templates SET archived_at = now() WHERE id = $1 AND archived_at IS NULL. Per D-07.

4. `func listTemplates(ctx context.Context, pool *pgxpool.Pool) ([]Template, error)` — SELECT from checklist_templates WHERE archived_at IS NULL ORDER BY created_at DESC. For each template, load sections (ordered), fields (ordered, with sub-steps nested via parent_field_id), schedules, and assignments. Return fully hydrated Template structs.

5. `func getTemplateByID(ctx context.Context, pool *pgxpool.Pool, templateID string) (*Template, error)` — Same as listTemplates but for a single template. Return nil if not found or archived.

**Submission CRUD:**

6. `func submitChecklist(ctx context.Context, pool *pgxpool.Pool, input SubmitChecklistInput, userID string) (string, error)` — Load template for snapshot (getTemplateByID). Check template not archived (return error if so — 409 conflict per D-14). Marshal template as JSON for template_snapshot. INSERT INTO checklist_submissions with ON CONFLICT (idempotency_key) DO UPDATE SET idempotency_key = EXCLUDED.idempotency_key RETURNING id (per D-15). Move draft responses (submission_id IS NULL, answered_by = userID, field_id in template's fields) to this submission by UPDATE SET submission_id = $1. Insert any additional responses from input.Responses. Insert fail notes from input.FailNotes. Return submission ID.

7. `func saveResponse(ctx context.Context, pool *pgxpool.Pool, fieldID string, value json.RawMessage, userID string) error` — INSERT INTO submission_responses (field_id, value, answered_by) VALUES ($1, $2, $3) ON CONFLICT ON CONSTRAINT submission_responses_draft_idx DO UPDATE SET value = EXCLUDED.value, answered_at = now(). This is the auto-save draft path (D-21, submission_id is NULL).

8. `func myChecklists(ctx context.Context, pool *pgxpool.Pool, userID string) ([]Template, []Submission, error)` — Two queries: (a) Templates assigned to user's role or user ID via template_assignments, filtered by schedule active_days containing today's DOW (EXTRACT(DOW FROM now())), not archived. (b) User's submissions from today (submitted_at >= current_date). Return both lists. Per D-22 today only.

9. `func myHistory(ctx context.Context, pool *pgxpool.Pool, userID string) ([]Submission, error)` — SELECT submissions WHERE submitted_by = userID ORDER BY submitted_at DESC LIMIT 50. Include template_name via JOIN.

**Approval CRUD:**

10. `func pendingApprovals(ctx context.Context, pool *pgxpool.Pool, userID string) ([]Submission, error)` — SELECT submissions WHERE status = 'pending' AND template is assigned to user as approver (via template_assignments WHERE assignment_role = 'approver'). Per D-23 group by template (ORDER BY template_id, submitted_at). Load responses, fail_notes, rejections for each.

11. `func approveSubmission(ctx context.Context, pool *pgxpool.Pool, submissionID string, reviewerID string) error` — UPDATE checklist_submissions SET status = 'approved', reviewed_by = $1, reviewed_at = now() WHERE id = $2 AND status = 'pending'. Return error if no rows affected.

12. `func rejectItem(ctx context.Context, pool *pgxpool.Pool, input RejectItemInput, rejectedBy string) error` — INSERT INTO submission_rejections. UPDATE checklist_submissions SET status = 'rejected', reviewed_by, reviewed_at WHERE id = input.SubmissionID.

**Draft cleanup:**

13. `func cleanupOldDrafts(ctx context.Context, pool *pgxpool.Pool) error` — DELETE FROM submission_responses WHERE submission_id IS NULL AND answered_at < current_date. Per pitfall 1 from RESEARCH.md — clean up yesterday's abandoned drafts.

Create `backend/internal/workflow/handler.go` in package `workflow`. Each handler is a closure over `*pgxpool.Pool` returning `http.HandlerFunc` — same pattern as `me.MeAppsHandler(pool)`. All handlers use `auth.UserFromContext(r.Context())` for the authenticated user. JSON responses use `json.NewEncoder(w).Encode(...)`. Errors return `{"error":"..."}` with appropriate status codes.

**Admin-only helper** (per D-11):
```go
func isAdmin(user *auth.User) bool {
    return user.Role == "admin" || user.IsSuperadmin
}
```

**Handlers (10 total, matching D-18 RPC style):**

1. `func ListTemplatesHandler(pool *pgxpool.Pool) http.HandlerFunc` — GET. Calls listTemplates(). Returns JSON array. Empty array if none.

2. `func CreateTemplateHandler(pool *pgxpool.Pool) http.HandlerFunc` — POST. Check isAdmin, return 403 `{"error":"forbidden"}` if not. Decode TemplateInput from body. Call insertTemplate(). Return 201 with `{"id":"..."}`.

3. `func UpdateTemplateHandler(pool *pgxpool.Pool) http.HandlerFunc` — PUT. Check isAdmin, return 403 if not. Get template ID from chi.URLParam(r, "id"). Decode TemplateInput. Call replaceTemplate(). Return 200 `{"ok":true}`.

4. `func ArchiveTemplateHandler(pool *pgxpool.Pool) http.HandlerFunc` — DELETE. Check isAdmin, return 403 if not. Get template ID from chi.URLParam(r, "id"). Call archiveTemplate(). Return 200 `{"ok":true}`.

5. `func MyChecklistsHandler(pool *pgxpool.Pool) http.HandlerFunc` — GET. Call myChecklists() with user.ID. Also call cleanupOldDrafts (fire-and-forget, log errors). Return JSON object `{"templates": [...], "submissions": [...]}`.

6. `func MyHistoryHandler(pool *pgxpool.Pool) http.HandlerFunc` — GET. Call myHistory(). Return JSON array.

7. `func SaveResponseHandler(pool *pgxpool.Pool) http.HandlerFunc` — POST. Decode SaveResponseInput. Call saveResponse() with user.ID. Return 204 No Content on success.

8. `func SubmitChecklistHandler(pool *pgxpool.Pool) http.HandlerFunc` — POST. Decode SubmitChecklistInput. Call submitChecklist(). If template archived error → return 409 `{"error":"template_archived"}`. Return 201 `{"id":"..."}`.

9. `func PendingApprovalsHandler(pool *pgxpool.Pool) http.HandlerFunc` — GET. Call pendingApprovals() with user.ID. Return JSON array.

10. `func ApproveSubmissionHandler(pool *pgxpool.Pool) http.HandlerFunc` — POST. Decode `{"submission_id":"..."}`. Call approveSubmission(). Return 200 `{"ok":true}`.

11. `func RejectItemHandler(pool *pgxpool.Pool) http.HandlerFunc` — POST. Decode RejectItemInput. Call rejectItem(). Return 200 `{"ok":true}`.

All handlers: set `Content-Type: application/json`. On decode error: 400. On internal error: 500 with `{"error":"internal_error"}` and `log.Printf`.
