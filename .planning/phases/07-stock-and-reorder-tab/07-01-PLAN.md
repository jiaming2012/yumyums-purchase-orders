  **1. Add MOCK_SALES constant** (per D-05) after MOCK_MENU_ITEMS. Weekly sales counts per menu item spanning the same date range as purchase events (Jan-Apr 2026). Structure as array of `{menuItemId, weekOf, qty}` objects. Create ~12-14 weeks of data. Vary quantities realistically:
  - Cheesesteak: 40-55/week (best seller)

  - Salmon Bowl: 15-25/week
  - Loaded Fries: 30-45/week

  **2. Add EXPANDED_STOCK_ITEMS state dict** (per code_context — reuse expand pattern from EXPANDED_EVENTS):

  **3. Implement calcStockLevels() function** (per D-04, D-06, D-07, D-08):

  Algorithm — sales-based consumption:
  a. For each MOCK_ITEM_GROUP, calculate `totalPurchased`:
     - Find all MOCK_PURCHASES where purchaseItemId maps to a MOCK_PURCHASE_ITEMS with this group's groupId
     - Sum qty values (this is total units purchased)

  b. Calculate `totalConsumed` using MOCK_SALES and MOCK_MENU_ITEMS ingredient ratios:
     - For each MOCK_MENU_ITEMS that includes this group in ingredientGroupIds:
       - Sum all MOCK_SALES qty for that menu item
       - Divide by number of ingredientGroupIds for that menu item (equal proportion assumption)
     - totalConsumed = sum of these across all menu items using this group

  c. Calculate `remaining = totalPurchased - totalConsumed` (floor at 0)

  d. Calculate `avgWeeklyConsumption`:
     - Total consumed / number of weeks in sales data range

  e. Determine stock level (per D-07):
     - If no purchase history for this group: level = 'unknown' (per D-08)
     - If remaining > avgWeeklyConsumption * 1: level = 'high'
     - If remaining > avgWeeklyConsumption * 0.5: level = 'medium'
     - Else: level = 'low'

  f. Return array of objects:
  ```javascript
  { groupId, groupName, tagIds, level, remaining, avgWeeklyConsumption,
    lastPurchaseDate, lastVendor, lastPrice, totalPurchased, totalConsumed }
  ```
  For lastPurchaseDate/lastVendor/lastPrice: find the most recent MOCK_PURCHASE for this group's items, look up its event date and vendor.

  **4. Wire into show() dispatch** — uncomment/add the ACTIVE_TAB===3 case in render():
  ```javascript
  if(ACTIVE_TAB===3)renderStock();
  ```
