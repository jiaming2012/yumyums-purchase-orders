- `setTextAnswer(tplId, fldId, value)` — set `completedItems[fldId] = { value: value, answeredBy: CURRENT_USER, answeredAt: new Date().toISOString() }`. Do NOT call `renderFillout()` (cursor jump prevention per Pitfall 2).

- `submitChecklist(tplId)` — set `fillState.responses[tplId].submitted = true`. Call `renderFillout()`.

**4. Temperature Input Handler (special):**

In the existing `input` event listener on #s1, add handling for `temp-input`:

- After re-render, the input will lose focus. To mitigate: after renderFillout(), find the temp input by `data-fld-id` and restore focus + cursor position.
