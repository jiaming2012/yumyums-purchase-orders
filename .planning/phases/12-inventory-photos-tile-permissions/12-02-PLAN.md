    Create backend/internal/photos/spaces.go:
    - SpacesConfig struct {AccessKey, SecretKey, Endpoint, Region, Bucket string}
    - NewSpacesPresigner(cfg SpacesConfig) (*s3.PresignClient, error) — uses aws-sdk-go-v2 with custom endpoint resolver for DO Spaces. MUST set UsePathStyle: true (DO Spaces requires path-style addressing, not virtual-hosted).
    - GeneratePresignedPutURL(ctx, presigner, bucket, key, contentType string, ttl time.Duration) (string, error) — wraps PresignPutObject
    - GeneratePresignedGetURL(ctx, presigner, bucket, key string, ttl time.Duration) (string, error) — wraps PresignGetObject
    - PublicURL(endpoint, bucket, key string) string — returns the permanent public URL: https://{bucket}.{region}.digitaloceanspaces.com/{key}

    Create backend/internal/photos/handler.go:
    - PresignUploadHandler(presigner *s3.PresignClient, bucket string) http.HandlerFunc
      - Accepts POST with JSON body: {path_prefix: "checklists" or "receipts", id: string, filename: string}
      - Builds key: "{path_prefix}/{id}/{filename}" (per D-02)
      - Generates presigned PUT URL with 15-minute TTL
      - Returns JSON: {url: presignedURL, object_key: key, public_url: permanentURL}
    - PresignGetHandler(presigner *s3.PresignClient, bucket string) http.HandlerFunc
      - Accepts GET with ?key= query param
      - Generates presigned GET URL with 1-hour TTL
      - Returns JSON: {url: presignedURL}

    Wire in main.go:
    - Add SpacesConfig to config loading from env vars: DO_SPACES_KEY, DO_SPACES_SECRET, DO_SPACES_BUCKET, DO_SPACES_REGION, DO_SPACES_ENDPOINT (default "https://{region}.digitaloceanspaces.com")
    - Initialize presigner in main() — if env vars missing, log warning and skip (photos endpoints return 503)
    - Add r.Route("/photos", func(r chi.Router) { r.Post("/presign", photos.PresignUploadHandler(...)); r.Get("/presign", photos.PresignGetHandler(...)) }) inside /api/v1 group

    Install new deps:
    ```
    cd backend && go get github.com/aws/aws-sdk-go-v2/service/s3 github.com/aws/aws-sdk-go-v2/config github.com/aws/aws-sdk-go-v2/credentials github.com/aws/aws-sdk-go-v2/service/s3/presigned
    ```
