    - "Discard Receipt" button (.btn-secondary)
    - On click: native confirm("Discard this receipt? It will be removed from the review queue.")

    - On success: remove card from pending list, decrement badge count

    6. CSS additions (inline):

    - .line-item-row: flex row with inputs
