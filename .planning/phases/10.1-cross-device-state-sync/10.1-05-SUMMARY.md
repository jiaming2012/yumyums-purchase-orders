## Decisions Made
- `showSyncToast` is separate from `showToast` — the existing `showToast` function uses `#toast` (styled as a dark action banner, positioned at bottom:24px), while sync toasts use `#sync-toast` (themed with `--card`/`--brd` for readability, positioned at bottom:70px above bottom nav area)
- `flashField` uses CSS class animation rather than inline style transitions so that animation properties are purely declarative and the offsetWidth reflow trick guarantees restart
- Dark mode `--sync-highlight` added to CSS variable block (not hardcoded in keyframe) so the flash color adapts automatically without JS
