   - In `main()`, implement the STATIC_DIR switching logic exactly per Pattern 1:
     ```go
     var staticFS fs.FS
     if dir := os.Getenv("STATIC_DIR"); dir != "" {
         // Dev: serve from disk — no rebuild needed for frontend changes
         log.Printf("Serving static files from disk: %s", dir)
         staticFS = os.DirFS(dir)
     } else {
         // Prod: serve from embedded FS (files baked into binary)
         log.Println("Serving static files from embedded FS")
         sub, err := fs.Sub(embeddedFS, "public")
         if err != nil {
             log.Fatalf("Failed to access embedded public dir: %v", err)
         }
         staticFS = sub
     }
     ```
     NOTE: `fs.Sub(embeddedFS, "public")` is needed because `//go:embed all:public` nests files under `public/` prefix. Sub-ing removes that prefix so `index.html` is served at `/` not `/public/index.html`.

2. Replace the fetch event listener (lines 18-22) with the partitioned version:
   ```js
   self.addEventListener('fetch', e => {
     const url = new URL(e.request.url);
     if (url.pathname.startsWith('/api/')) {
       // Network-first: API calls always go to network; no caching
       e.respondWith(fetch(e.request).catch(() => new Response('{"error":"offline"}', {
         status: 503,
         headers: {'Content-Type':'application/json'}
       })));
       return;
     }
     // Cache-first: static assets served from cache; fallback to network
     e.respondWith(
       caches.match(e.request).then(r => r || fetch(e.request).catch(() => caches.match('./index.html')))
     );
   });
   ```

Do NOT change the install or activate event listeners -- they remain as-is. The ASSETS array also stays unchanged.
