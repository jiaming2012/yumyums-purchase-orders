    - If no LOCKED_PO: show stub "No locked PO yet. Orders lock at the weekly cutoff."

        ```javascript
        var vendorGroups = {};

          var vname = item.vendor_name || 'Unassigned';

          vendorGroups[vname].push(item);
        });
        ```
      - Each vendor section: header with vendor name + item count + estimated total

      - Bottom: "Approve & Create Shopping List" button (admin-only, per D-10)
        - Style: green accent, full-width
        - On 409 from approve endpoint: show message "Complete last week's shopping list before approving this one" (per D-11)
        - On success: reload page to show Shopping tab with new list
